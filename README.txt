README - ITALIANO

C_GPS2KML - http://gnss-info.blogspot.com

Il programma converte files in KML/KMZ per essere visualizzati da Google Earth.
Per i formati supportatie le opzioni del programma, per favore fai riferiemnto alla documentazione disponibile online a http://gnss-info.blogspot.com/p/cgps2kml.html

Il contenuto del file .zip e il seguente:
C_GPS2KML.exe   - Programma principale eseguibile (eseguibile anche da CLI)
GarminSymbols.txt - file di definizione simboli e icone custom per conversione dei Waypoints dei files GPX Garmin 
Icons.txt - Definizione delle icone (per la visualizzaione dei punti).
Styles.txt - Definizione degli stili.
Sym_AsterixBN.bmp, Sym_GreenCross.bmp, Sym_RedSquare.bmp - Esempio di symboli per GarminSymbols.txt
Example_car.nmea - Esempio di file registrato con un logger GPS.
Example_car.kmz - Esempio di file convertito con C_GPS2KML per mostrare alcune delle possbilita disponibili nella visualizzazione con Google Earth.

README - ENGLISH

C_GPS2KML - http://gnss-info.blogspot.com

The program converts files into KML/KMZ for Google Earth visualization.
For more informations,please refer to the documentation at http://gnss-info.blogspot.com/p/cgps2kml-eng.html

The content of the .zip file is the following
C_GPS2KML.exe   - Main program (CLI usage is supported)
GarminSymbols.txt - file containing the definitions of the symbols and custom icons that can be used to convert Waypoints from Garmin GPS files.
Icons.txt - Icons definition (for points visualization).
Styles.txt - Styles definitions.
Sym_AsterixBN.bmp, Sym_GreenCross.bmp, Sym_RedSquare.bmp - Examples of symbols for GarminSymbols.txt
Example_car.nmea - Example of a file logged with a GPS receiver
Example_car.kmz - Example of a file converted using C_GPS2KML to show some of the features available in the Google Earth Visualization.
